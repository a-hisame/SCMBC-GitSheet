github事始め
GitHubのプロジェクトを作成
空のプロジェクトを（ローカルに）クローンgit clone git@github.com:a-hisame/Project.git
何かファイルを作成してコミット
  • git add .
  • git commit -m 'commit comment'
ローカルからGitHubにpush git push origin master
GitHub側にdevelopment branchを作成 git push origin master:development
ローカルにGitHubのブランチをコピー git branch development origin/development
ブランチを移動git checkout development
ファイルを作る。add/commitgit add * / git commit *
コミットをpushgit push origin development
コンフリクト！ もう一度pullgit pull
オートコンフリクト修正に失敗した場合はファイルを直す git add [conflict file] / git commit
バージョン管理
変更をコミットするgit commit -a
ひとつ前のコミットをチェックアウトするgit co HEAD^
変更を元に戻すgit reset HEAD --hard
